# Generation of Data for visualization in Flutter App

Idea: Generate data as Dart Stream access via REST API.

The later the output of the Flutter App looks like his ["Sample usage of the data being generated by such an API"]( https://www.youtube.com/watch?v=40JgkY5b3W0)


## Instantiation of the Frame Object

GenerateData gd = GenerateData(\<List of parameters\>);

This constructor requires the following parameters:

| Parameter  | Type | Description |
|------------|------|-------------|
|Title       | String | The title of the data (markdown format) |
|Subtitle    | String | The subtitle of the data (markdown format) |
|Start Date  | DateTime | The start date of the data |
|End Date    | DateTime | The end date of the data |
|DataStrips  | List<DataStrip> | Number of data strips to generate |


## Data strip

| Parameter  | Type   | Description |
|------------|--------|-------------|
| Name        | String | The name of the data strip |
| Start value | double | The start value of the data |
| End value   | double | The end value of the data |
| Interval    | (String, int) | The interval between data points: as set of interval type and interval value (size). See below |
| Value       | double | The current value of the data strip |
| Icon        | URL (String) | Link to he icon of the data strip |
| Color       | Color  | The color of the data strip |
| Height      | double | The height of the data strip |


### Interval Options

|Interval | Code | Default Value |
|---------|------|---------------|
|hours     | H    | 1     |
|days      | D    | 1     |
|weeks     | W    | 1     |
|months    | M    | 1     |
|quarters  | Q    | 1     |
|years     | Y    | 1     |


## Methods of Frame Object

| Method | Parameters | Return Type | Description |
|--------|------------|-------------|-------------|
| getStrips | optional: strip number | int | Returns JSON of a specified strip, if not specified, return all strips as a list, sorted by value (decreasing) |
| startStream | optional: maxIterations | int | The maximum number of iterations to perform. If not specified, the stream will continue indefinitely. | Starts the stream of JSON data strips |
| stopStream | - | void | Stops the stream of JSON data strips |
| resetAll   | - | void | Resets all strips to their initial state |


## Methods of Strip Object

| Method | Parameters | Return Type | Description |
|--------|------------|-------------|-------------|
| reset | - | void | Resets the strip to its initial state |
| setStartValue | double | void | Sets the start value of the strip |
| setEndValue | double | void | Sets the end value of the strip |
| setInterval | (String, int) | void | Sets the interval of the strip |
| setIcon | String | void | Sets the icon of the strip |
| setColor | Color | void | Sets the color of the strip |
| toJson | - | String | Returns the JSON of the strip |
